Markdown 数学公式和符号

发表于 2020-09-25 分类于 Markdown 阅读次数：

因为经常需要查一些数学符号的写法，有点麻烦，也没有找到一个比较全面的博客，所以决定自己整理一份，方便查找。

显示数学公式或符号

$ 数学公式或符号 $

上 / 下标

  • 上标：^
    • 比如：x^2
  • 下标：_
    • x_2

占位符

  • 两个 quad 空格：\qquad
  • quad 空格：\quad
  • 空格：\
  • 紧贴：\!

四则运算

  • 加：+
  • 减：-
  • 乘：\times
  • 点乘：\cdot
  • 星乘：\ast 或 *
  • 除：\div 或 /
  • 加减：\pm
  • 减加：\mp

高级运算

  • 分式：\frac{ 分子 }{ 分母 } 或 { 分子 }\over{ 分母 }
  • 绝对值：| |
  • 平均数：\overline{ 算式 } (\overline{xyz})
  • 开方：
    • 二次：\sqrt (\sqrt{x})
    • 多次：\sqrt[开方数]{ 被开方数 } (\sqrt[3]{x+y})
  • 对数：\log (\log(x))
  • 极限：\lim (\lim^{x \to \infty}_{y \to 0}\frac{x}{y})
