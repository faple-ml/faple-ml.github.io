Markdown 数学公式和符号

发表于 2020-09-25 分类于 Markdown 阅读次数：

因为经常需要查一些数学符号的写法，有点麻烦，也没有找到一个比较全面的博客，所以决定自己整理一份，方便查找。

显示数学公式或符号

$ 数学公式或符号 $

上 / 下标

  • 上标：^
  • 下标：_

占位符

  • 两个 quad 空格：\qquad
    • 比如：x \qquad y 为 $x \qquad y$
  • quad 空格：\quad
    • 比如：x \quad y 为 $x \quad y$
  • 空格：\
    • 比如：x \ y 为 $x \ y$
  • 紧贴：\!
    • 比如：x ! y 为 $x ! y$

四则运算

  • 加：+
  • 减：-
  • 乘：\times
  • 点乘：\cdot
  • 星乘：\ast 或 *
  • 除：\div 或 /
  • 加减：\pm
    • 比如： $x \pm y$
  • 减加：\mp
    • 比如：$x \mp y$

高级运算

  • 分式：\frac{ 分子 }{ 分母 } 或 { 分子 }\over{ 分母 }
    • 比如：\frac{x}{y} 或 {x} \over {y} 为 $\frac{x}{y}$
  • 绝对值：| |
  • 平均数：\overline{ 算式 }
    • 比如：\overline{xyz} 为 $\overline{xyz}$
  • 开方：
    • 二次：\sqrt
    • 多次：\sqrt[开方数]{ 被开方数 }
      • 比如：\sqrt[3]{x+y} 为 $\sqrt[3]{x+y}$
  • 对数：\log
    • 比如：\log(x) 为 $\log(x)$
  • 极限：\lim
    • 比如：\lim^{x \to \infty}_{y \to 0}\frac{x}{y} 为 $\lim^{x \to \infty}_{y \to 0}\frac{x}{y}$
  • 求和：\sum 或 \displaystyle \sum
    • 比如：\sum^{x \to \infty}_{y \to 0}\frac{x}{y} 为 $\sum^{x \to \infty}_{y \to 0}\frac{x}{y}$
    • 或者 \displaystyle \sum^{x \to \infty}_{y \to 0}\frac{x}{y} 为 $\displaystyle \sum^{x \to \infty}_{y \to 0}\frac{x}{y}$
  • 积分：\int 或 \displaystyle \int
    • 比如：\int^{\infty}_{0}{xdx} 为 $\int^{\infty}_{0}{xdx}$
  • 微分（偏导）：\partial
    • 比如：\frac{\partial x}{\partial y} 为 $\frac{\partial x}{\partial y}$
  • 矩阵：\begin{matrix} \end{matrix}
    • 比如：\left[\begin{matrix} 1 &2 &\cdots &4\5 &6 &\cdots &8\vdots &\vdots &\ddots &\vdots\13 &14 &\cdots &16\end{matrix} \right] 为 $\left[\begin{matrix} 1 &2 &\cdots &4\5 &6 &\cdots &8\vdots &\vdots &\ddots &\vdots\13 &14 &\cdots &16\end{matrix} \right]$

逻辑运算

  • 等于：=
  • 大于：>
  • 小于：<
  • 大于等于：\geq
  • 小于等于：\leq
  • 不等于：neq
  • 不大于等于：\ngeq 或 \not\geq
    • 比如：x+y \ngeq z 为 $x+y \ngeq z$
  • 不小于等于：\nleq 或 \not\leq
  • 约等于：\approx
    • 比如：x+y \approx z 为 $x+y \approx z$
  • 不约等于：\not\approx
  • 近似等于：\simeq
    • 比如：x+y \simeq z 为 $x+y \simeq z$
  • 全等：\cong
    • 比如：x+y \cong z 为 $x+y \cong z$
  • 恒等于：\equiv
    • 比如：x+y \equiv z 为 $x+y \equiv z$

集合运算

  • 属于：\in
    • 比如：$x \in y$
  • 不属于：\notin 或 \not\in
    • 比如：$x \notin y$
  • 子集：\subset 或 \supset
    • 比如：$x \subset y$ 或 $x \supset y$
  • 非子集：\not\subset 或 \not\supset
  • 真子集：\subseteq 或 \supseteq
    • 比如：$x \subseteq y$ 或 $x \supseteq y$
  • 非真子集：\subsetneq 或 \supsetneq
  • 并集：\cup
    • 比如：$x \cup y$
  • 交集：\cap
    • 比如：$x \cap y$
  • 差集：\setminus
    • 比如：$x \setminus y$
  • 同或：\bigodot
    • 比如：$x \bigodot y$
  • 同与：\bigotimes
    • 比如：$x \bigotimes y$
  • 实数集：\mathbb{R}
    • 比如：$\mathbb{R}$
  • 自然数集：\mathbb{Z}
    • 比如：$\mathbb{Z}$
  • 空集：\emptyset
    • 比如：$\emptyset$

数学符号

  • 无穷：\infty
    • 比如：$\infty$
  • 虚数：\imath 或 \jmath
    • 比如：$\imath$ 或 $\jmath$

戴帽符号

  • $\hat{a}$：\hat{a}
  • $\check{a}$：\check{a}
  • $\breve{a}$：\breve{a}
  • $\tilde{a}$：\tilde{a}
  • $\bar{a}$：\bar{a}
  • $\vec{a}$：\vec{a} （矢量）
  • $\acute{a}$：\acute{a}
  • $\grave{a}$：\grave{a}
  • $\mathring{a}$：\mathring{a}
  • $\dot{a}$：\dot{a}
  • $\ddot{a}$：\ddot{a}

箭头符号

  • $\uparrow$：\uparrow
  • $\Uparrow$：\Uparrow
  • $\downarrow$：\downarrow
  • $\Downarrow$：\Downarrow
  • $\leftarrow$：\leftarrow
  • $\Leftarrow$：\Leftarrow
  • $\rightarrow$：\rightarrow
  • $\Rightarrow$：\Rightarrow
  • $\longleftarrow$：\longleftarrow
  • $\Longleftarrow$：\Longleftarrow
  • $\longrightarrow$：\longrightarrow
  • $\Longrightarrow$：\Longrightarrow

省略号

  • 底端对齐：\ldots
    • 比如：1,2,\ldots,n 为 $1,2,\ldots,n$
  • 中线对齐：\cdots
    • 比如：x_1^2 + x_2^2 + \cdots + x_n^2 为 $x_1^2 + x_2^2 + \cdots + x_n^2$
  • 竖直对齐：\vdots
    • 比如：$\vdots$
  • 斜对齐：\ddots
    • 比如：$\ddots$

括号

  • 各种大小的小括号：
    • $()$：()
    • $\big(\big)$：\big(\big)
    • $\Big(\Big)$：\Big(\Big)
    • $\bigg(\bigg)$：\bigg(\bigg)
    • $\Bigg(\Bigg)$：\Bigg(\Bigg)
  • 中括号：[]
  • 大括号：\{\}
  • 自适应括号：\left \right
    • 比如：\left(\frac12\right) 为 $\left(\frac12\right)$

  • 组合公式：{ 上位公式 \choose 下位公式 } 或 { 上位公式 \atop 下位公式 }

    • 比如：{n+1 \choose k}={n \choose k}+{n \choose k-1} 为 ${n+1 \choose k}={n \choose k}+{n \choose k-1}$
    • 或者 \sum_{k0,k_1,\ldots>0 \atop k_0+k_1+\cdots=n}A\{k0}A\{k1}\cdots 为 $\sum{k0,k_1,\ldots>0 \atop k_0+k_1+\cdots=n}A{k0}A{k_1}\cdots$
